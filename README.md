# DELTA-BIT

## Installing
copy DELTA-BIT in your computer and insert its path in the .bashrc file. Suppose you saved DELTA-BIT in path/to/DELTA-BIT type in your home:
>printf "\n%s\n" "export DELTA_BIT=path/to/DELTA-BIT" >> ~/.bashrc

>source ~/.bashrc
